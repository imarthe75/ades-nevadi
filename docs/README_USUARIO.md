Guía breve — Instituto Nevadi (ADES)

¿Para qué sirve este sistema?

ADES (Aplicación de Gestión Escolar del Instituto Nevadi) es la plataforma centralizada para las operaciones académicas y administrativas del plantel. Está diseñada para gestionar y facilitar:

- Matrícula, inscripciones y reinscripción masiva.
- Control de asistencia y libretas de calificaciones.
- Emisión y verificación de certificados digitales con firma y QR.
- Gestión de materias, planes de estudio y rúbricas.
- Evaluación docente 360° y reportes de desempeño.
- Comunicación con padres y notificaciones.

Para quién está pensado

- Directores y administrativos: gestionar ciclos, planteles, aprobaciones y procesos masivos.
- Coordinadores y docentes: planear materias, subir tareas, calificar y usar rúbricas.
- Alumnos y padres: consultar calificaciones, certificados y comunicados.
- Equipo técnico: acceder a paneles de administración, auditoría y operaciones.

Principales funcionalidades (en lenguaje no técnico)

- Panel de inicio personalizado: muestra el ciclo activo, plantel y accesos rápidos.
- Catálogos maestros: niveles educativos, ciclos, grados y materias para usar en todos los módulos.
- Reinscripción masiva: validar y aprobar promociones por ciclo con reportes por plantel.
- Certificados digitales: emitir PDF con QR verificable y firmar con llave institucional.
- Evaluación docente 360°: crear evaluaciones, calificar por criterios y obtener resúmenes por profesor.
- Rúbricas y planificación: crear rúbricas por materia y usar plantillas para evaluación.
- Seguridad: inicio de sesión mediante el sistema institucional (SSO/OIDC) y permisos por rol.

Garantías de integridad y concurrencia

- El sistema está diseñado para múltiples usuarios simultáneos: profesores, administrativos y padres pueden trabajar a la vez.
- Operaciones críticas (p. ej. reinscripción masiva o cierres de ciclo) se ejecutan como procesos transaccionales y/o tareas en segundo plano para evitar inconsistencias.
- Existe trazabilidad y auditoría para cambios importantes (quién modificó qué y cuándo).

Qué esperar en la práctica (flujo típico)

1. Login: ingresa con la cuenta institucional (SSO). Si no tienes acceso, contacta a soporte.
2. Selecciona el ciclo y plantel (si aplica); la barra superior conserva tu contexto.
3. Usa los catálogos para buscar materias, grados o niveles cuando configures actividades.
4. Profesores: crea tareas, sube evidencias y califica. Usa las rúbricas para estandarizar calificación.
5. Administrativos: ejecuta reinscripción masiva con pasos de validación y aprobación.
6. Para certificados: genera, firma y descarga el PDF con QR de verificación pública.

Soporte y ayuda

- Si algo no aparece en tu pantalla (p. ej. listados vacíos o errores 401/404),:
  - Verifica que has iniciado sesión correctamente y que tu rol tiene permisos.
  - Refresca la página y vuelve a intentar la acción.
  - Revisa la sección de ayuda integrada o contacta al administrador con la descripción del error y una captura de pantalla.

Buenas prácticas para usuarios

- Evita realizar acciones masivas (ej. aprobar promos) durante horas pico si no es urgente.
- Siempre espera la confirmación del sistema antes de cerrar una ventana durante procesos largos (exportaciones, firmas, validaciones).
- Reporta incongruencias en calificaciones o datos personales inmediatamente al administrativo responsable.

Política de privacidad y uso de datos

- La plataforma almacena datos académicos y personales mínimos necesarios para operación escolar.
- Los accesos y cambios quedan registrados para auditoría.
- El uso de los datos se gestiona conforme a la normativa institucional y de protección de datos aplicable.
